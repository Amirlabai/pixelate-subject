Keyboard shortcuts
  • Ctrl + wheel — zoom in/out on the image
  • Ctrl + 0 — reset zoom to fit
  • Ctrl + + / Ctrl + − — zoom in/out
  • Ctrl + Z — undo brush stroke
  • Wheel — pan (when zoomed)
  • Shift + wheel — pan sideways
